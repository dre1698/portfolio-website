Pirate Island

Tool: Autodesk Maya

This 3D model was created as a stylized environment scene in Maya. It features a low-poly pirate ship with cannons, a small treasure island, palm tree, and surrounding props.

Focus:
The project was designed as a creative exercise in low-poly modeling and composition. Special attention was given to silhouette, visual balance, and the use of color to create a cohesive scene.

Outcome:
This project helped strengthen my skills in environment layout, modeling efficiency, and stylized 3D aesthetics.
